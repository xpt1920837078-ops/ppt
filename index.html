数据库设计

Database Design

数据库设计（ER图）

Database Design - Entity Relationship

👤 读者表 reader

  • 🔑 id（证件号）
  • name（姓名）
  • status（状态）
  • contact（联系方式）
1:N ➡

📖 借阅记录表 loan

  • 🔑 id（记录ID）
  • 🔗 reader_id（FK）
  • 🔗 book_isbn（FK）
  • 🔗 worker_id（FK）
  • borrow_date（借书日期）
  • due_date（应还日期）
  • return_date（还书日期）
  • fine（罚款金额）
⬅ N:1

📚 图书表 book

  • 🔑 isbn（ISBN）
  • title（书名）
  • author（作者）
  • publisher（出版社）
  • price（价格）

👷 工作人员表 worker

  • 🔑 worker_id
  • name（姓名）
  • phone（电话）

👤 用户表 user

  • 🔑 username
  • password（密码）
  • role（角色）
关系说明：读者、图书、工作人员与借阅记录之间都是1:N关系，通过外键约束保证数据完整性

核心代码实现

Core Code Implementation

核心代码实现 - 数据库工具类

Core Code - DBUtil Database Utility

DBUtil.java - 单例模式管理数据库连接

public class DBUtil { private static Connection conn = null; /** * 获取数据库连接 * 采用单例模式，确保整个应用只有一个数据库连接实例 */ public static Connection getConnection() throws SQLException { if (conn == null || conn.isClosed()) { Class.forName("org.sqlite.JDBC"); // 加载驱动 conn = DriverManager.getConnection("jdbc:sqlite:library.db"); conn.createStatement().execute("PRAGMA foreign_keys = ON"); // 启用外键 } return conn; } }

LoanDAO.java - 借阅记录创建（参数化查询）

public boolean createLoan(Loan loan) { String sql = "INSERT INTO loan (reader_id, book_isbn, worker_id, borrow_date, due_date) VALUES (?, ?, ?, ?, ?)"; try (Connection conn = DBUtil.getConnection(); PreparedStatement pstmt = conn.prepareStatement(sql)) { pstmt.setString(1, loan.getReaderId()); pstmt.setString(2, loan.getBookIsbn()); pstmt.setString(3, loan.getWorkerId()); pstmt.setString(4, loan.getBorrowDate()); pstmt.setString(5, loan.getDueDate()); return pstmt.executeUpdate() > 0; } }

核心代码实现 - 条形码生成

Core Code - Barcode Generation

BarcodeUtil.java - ZXing条形码生成（加分项）

public static BufferedImage generateBarcode(String isbn) { try { Code128Writer writer = new Code128Writer(); Map<EncodeHintType, Object> hints = new HashMap<>(); hints.put(EncodeHintType.CHARACTER_SET, "UTF-8"); // 生成位矩阵 BitMatrix bitMatrix = writer.encode( isbn, BarcodeFormat.CODE_128, 300, 80, hints); // 转换为BufferedImage return MatrixToImageWriter.toBufferedImage(bitMatrix); } catch (WriterException e) { e.printStackTrace(); return null; } }
技术要点：使用ZXing库的Code128编码格式，支持数字、字母、符号混合编码 | 生成300x80像素的条形码图片 | 可作为JLabel图标显示
